Diese Website wird über GitHub Pages bereitgestellt, was bedeutet, dass ein globales Verteilungsnetzwerk (CDN) genutzt wird, um Inhalte zuverlässig anzuzeigen. Beim Besuch der Website können bestimmte nicht-personenbezogene Daten, wie Ihre IP-Adresse und Informationen zur Seitennutzung, über dieses Netzwerk übertragen werden, um die ordnungsgemäße Funktion der Website sicherzustellen. Weitere Informationen finden Sie im Impressum

Bitte bestätigen Sie, dass Sie dem zustimmen und dass wir ein Consent-Cookie verwenden dürfen, das für die Dauer von einem Tag gespeichert wird. Dieses Cookie ermöglicht es uns, Ihre Zustimmung zur Nutzung nicht-personenbezogener Daten und zur Funktionsfähigkeit der Website zu speichern.
