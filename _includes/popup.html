Information zur Nutzung von GitHub Pages
Diese Website wird über GitHub Pages bereitgestellt und nutzt ein globales Verteilungsnetzwerk (CDN). Beim Besuch der Seite können einige nicht-personenbezogene Daten (z. B. Ihre IP-Adresse und grundlegende Nutzungsdaten) über dieses Netzwerk übertragen werden. Dies ist notwendig, damit die Website ordnungsgemäß funktioniert. Weitere Informationen finden Sie im Impressum.

Essentielles Cookie
Wir verwenden ein essentielles Cookie, um Ihre Präferenz zur Datenverarbeitung auf dieser Website zu speichern. Dieses Cookie ist notwendig für die Funktionalität des Cookie-Hinweises und wird für 24 Stunden gespeichert. Es enthält keine personenbezogenen Daten.

Bitte bestätigen Sie, dass Sie diese Information zur Kenntnis genommen haben. Diese Bestätigung bezieht sich ausschließlich auf das essentielle Cookie und die Nutzung von GitHub Pages – und stellt keine Zustimmung zur Verwendung weiterer Cookies oder zur Verarbeitung personenbezogener Daten dar.
Weitere Cookies werden auf dieser Website nicht verwendet.
